# User-level instructions for Codex CLI

Personal preferences and durable guidance that should apply to every Codex
session. Checked in — do not put secrets or per-project context here.

## Language

Reply to the user in the language they write in (e.g. Japanese in, Japanese
out). Everything else — code, inline comments, docstrings, commit messages,
PR descriptions, documentation, file and identifier names — must be in
English, regardless of the conversation language.
